# Decision Trees and Random Forest Project

## Description
This project applies Decision Tree and Random Forest models to predict credit policy approval using loan data.

## Dataset  📊
- loan_data.csv

## Models Used ⚙️
- Decision Tree
- Random Forest

## Results 📈 
- Decision Tree Accuracy: 0.98
- Random Forest Accuracy: 0.99

## Conclusion
Random Forest performed better because it reduces overfitting and improves accuracy.
